> **---- HAND SWITCH MOD ----**
> 
> This mod allows you to bind on any key switching main arm.
>
> To bind key you need to go Options -> Controls -> Key binds -> Find **"SWITCH HAND"** tab and option **"Switch Hand"** and bind any key ( i recommend button **r** )
